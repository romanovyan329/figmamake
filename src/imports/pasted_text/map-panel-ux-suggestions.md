Сейчас у тебя справа три кнопки: **+**, **фильтр**, **иконка маршрута**. Логика правильная, но для морского приложения лучше сделать их **более очевидными по действиям**. Вот как это обычно делают в навигационных приложениях.

---

# Правая панель кнопок (сверху вниз)

## ➕ Add Point / Add Marker

Создание новой точки на карте.

Что происходит при нажатии:

Появляется меню:

• Add Marina
• Add Anchorage
• Add Fuel Station
• Add Danger Zone
• Add Custom Point

После выбора пользователь ставит точку на карте и добавляет:

• название
• тип
• заметку
• координаты

Это одна из **главных функций карты**.

---

## 🔎 / Funnel — Filters

Фильтрация точек на карте.

Открывает панель:

Show:

✓ Marinas
✓ Ports
✓ Anchorages
✓ Fuel Stations
✓ Danger Zones
✓ My Points

Также можно добавить:

• Distance radius
• Only nearby points

Это помогает **очистить карту от лишнего**.

---

## ➤ Routes / Navigation

Работа с маршрутами.

Открывает:

• список маршрутов
• создать новый маршрут
• построить маршрут от текущей точки

Также можно добавить быстрый режим:

**Start Route**

когда пользователь начинает навигацию.

---

# Правильный порядок кнопок

Сейчас порядок нормальный, но лучше сделать:

```
+
Routes
Filters
```

или

```
Add point
Routes
Filters
```

Почему:

1️⃣ Добавление точек — самое частое действие
2️⃣ Маршрут — второе по важности
3️⃣ Фильтр используется реже

```

Add Point

Save Location
```

# Еще одна очень полезная кнопка (рекомендую)

Добавить:

📍 **My Location**

Она центрирует карту на лодке или GPS.

Навигационные приложения почти всегда имеют её.

Тогда панель будет:

```
+
Routes
Filters
My location
```

---

# Цвета кнопок

Сейчас они темные — это нормально, но можно сделать:

• primary action (+) — cyan
• остальные — dark navy
