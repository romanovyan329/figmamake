## PROMPT

Design a **mobile screen called "Vessel Settings"** for a marine utility app.

This screen opens when the user taps the **gear icon on the "My Vessel" page**.

The purpose of the screen is to **edit basic boat information**.

Use a **dark marine theme** consistent with the rest of the app:

deep navy background
dark blue cards
cyan accent color
soft white text

The interface should be **clean, minimal, and practical**.

Avoid complex settings — this is an **MVP version**.

---

## Screen layout

Title at top:

Vessel Settings

Below show a **simple form with editable fields**.

Fields:

Boat Name
Vessel Type (dropdown)

Options:
Motor Yacht
Sailing Yacht
Boat
Catamaran

Manufacturer / Model

Year

Length (meters)

Max Speed (knots)

Fuel Type (dropdown)

Options:
Diesel
Petrol
Electric

---

## UI structure

Each field should appear as a **clean input card**.

Example:

Boat Name
[ Sea Spirit ]

Length
[ 12.5 m ]

Max Speed
[ 28 knots ]

---

## Bottom actions

At the bottom include two buttons:

Save Changes (primary button)

Cancel

---

## UX rules

Keep the screen **simple and focused**.

Do not include:

engine service configuration
maintenance intervals
advanced vessel data
cloud syncing

Only allow users to **edit the core vessel information** used in the app.
