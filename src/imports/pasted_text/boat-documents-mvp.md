# PROMPT

Design a mobile screen for the **Boat Documents** section of a marine utility app.

The screen should allow users to **store, view, add, and delete important boat documents**.

The interface should be minimal and practical, following a **dark marine theme** similar to the rest of the app.

Use colors like:

deep navy background
dark blue cards
cyan accent
soft white text

Avoid complex layouts — this is a **minimal MVP version**.

---

## Main screen layout

Title at top:

Boat Documents

Under the title show a **list of uploaded documents**.

Each document card should display:

Document name
Document type (Registration, Insurance, Certificate, etc.)
Issue date
Expiration date (if available)
File type icon (PDF or image)

Example card layout:

Boat Registration
Issued: 12 May 2023
Expires: 12 May 2026
PDF

Each card should have a small action menu with:

View
Delete

---

## Add Document button

At the bottom or top right include a primary button:

Add Document

When tapped, open a simple **Add Document form**.

---

## Add Document Form

Fields:

Document Name
Document Type (dropdown)

Options:

Registration
Insurance
Certificate
License
Other

Issue Date
Expiration Date (optional)

Upload File

Allow:

PDF
Image
Photo from camera

Buttons:

Cancel
Save Document

---

## Document View

When a user taps a document, open a **simple viewer screen** showing the document preview (PDF or image).

Include actions:

Delete Document
Close

---

## MVP Rules

Keep the interface **simple and functional**.

Do not include:

sharing
cloud integrations
advanced permissions
collaboration

Focus only on:

view
add
delete
preview documents.
