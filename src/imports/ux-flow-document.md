Отлично. Сейчас это красивый UI-шаблон, но нет логики переходов. Ниже — чёткий UX-флоу, чтобы приложение ощущалось как профессиональный инструмент, а не список карточек.

Я распишу по блокам:

---

# 1️⃣ Главный экран (Home / Tournament Navigator)

### Что это?

Обзор + быстрый доступ к действиям.

### Интерактивные зоны:

#### 🔎 Поиск (Search bar)

Тап → открывается **Full Search Screen**, не просто поле.

Экран поиска:

* Поле ввода (автофокус)
* Быстрые фильтры (чипы):

  * Country
  * Discipline
  * Date
  * Prize Pool
  * Status (Open / Upcoming / Closed)
* История последних поисков
* Результаты в реальном времени

UX-правило:
Поиск должен искать:

* По названию турнира
* По городу
* По стране
* По дисциплине

---

#### 🔽 Иконка фильтра

Тап → открывается **Bottom Sheet Filter Panel**

Фильтр:

* Country (мультивыбор)
* Discipline
* Date range
* Prize pool (ползунок €)
* Status
* Format (Team / Individual)

Кнопки:

* Reset
* Apply Filters

После Apply:
Home обновляется.

---

# 2️⃣ Карточка турнира — “Details”

При нажатии на карточку ИЛИ кнопку Details:

Открывается **Tournament Details Screen**

Структура:

### 1. Header

* Название
* Статус (Open / Upcoming)
* Фото водоёма

### 2. Основная информация

* Локация (кликабельная → открывает экран водоёма)
* Даты
* Количество спортсменов
* Призовой фонд
* Дисциплина
* Формат

### 3. Описание

* Регламент
* Лимиты
* Зоны
* История турнира

### 4. Блок “My Participation”

Если турнир не добавлен:
→ Кнопка: Add to My Season

Если добавлен:
→ View My Entry

---

# 3️⃣ Регистрация (Register)

Важно: это не реальная регистрация, а внутренняя фиксация участия.

При нажатии Register:

Открывается **Registration Sheet**

Поля:

* Individual / Team
* Выбор команды (если есть)
* Цель (Optional)
* Заметки
* Планируемая дисциплина (если несколько)

Кнопка:
Confirm Participation

После подтверждения:

* Турнир добавляется в My Season
* Кнопка Register меняется на “Manage Entry”

---

# 4️⃣ My Season (через Stats или отдельный экран)

Турнир после регистрации попадает в:

📊 Stats → Current Season

Внутри турнира можно:

* Ввести вес по дням
* Добавить Big Fish
* Внести финальное место
* Добавить заметки
* Добавить погодные условия

После завершения:
Турнир становится частью аналитики.

---

# 5️⃣ Экран водоёма (если тап по локации)

Lake / Water Screen:

* Фото
* Страна
* Тип
* Средняя глубина
* Турниры, проходившие здесь
* Моя статистика на этом водоёме (если есть)

Это создаёт профессиональный уровень.

---

# 6️⃣ View All (Upcoming)

Кнопка View All →
Открывается полноценный список турниров:

Сортировка:

* By Date
* By Prize Pool
* By Country
* By Status

# 7️⃣ Нижнее меню

Обзор + ближайшие турниры

# 8️⃣ Поведение статусов

Open → можно добавить в сезон
Upcoming → можно добавить
Closed → можно только просмотреть

Если турнир уже в сезоне:

* Register скрывается
* Показывается статус “Added”

---

# 9️⃣ UX принципы (чтобы было профессионально)

1. Никаких всплывающих подтверждений без смысла
2. Всегда понятно:

   * Я участвую?
   * Я добавил?
   * Это часть моего сезона?
3. Цвета статусов:

   * Open → зелёный
   * Upcoming → синий
   * Closed → серый
4. После любого действия — лёгкий micro-feedback (иконка ✓)

---
